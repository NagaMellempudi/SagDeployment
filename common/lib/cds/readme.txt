My webMethods Server supports CDS remote access using Universal Messaging as the transport:

Your client application will require the following jar files in the classpath:

common/lib/wm-directory.jar
common/lib/wm-mws-directory-um.jar
common/lib/wm-g11nutils.jar
common/lib/wm-scg-core.jar
common/lib/ext/log4j.jar
common/lib/ext/slf4j-api.jar
common/lib/ext/slf4j-log4j.jar
common/lib/glassfish/gf.jackson-core-asl.jar
common/lib/glassfish/gf.jackson-mapper-asl.jar
common/lib/glassfish/gf.javax.jms.jar
common/lib/nClient.jar
common/lib/nJMS.jar
